Taller Mecánico Solana 🛠️

Contrato inteligente desarrollado con Anchor para gestionar reparaciones mecánicas en la blockchain de Solana.

Funcionalidades
Crear Reparación: Registra un vehículo con su descripción y costo inicial.

Actualizar Costo: Permite modificar el presupuesto de una reparación activa.

Terminar Reparación: Cierra el registro y libera la renta (SOL) a la billetera del usuario.
